# Christoffel Private Chef Menu Management App

## рџ“± Project Overview

The **Christoffel Private Chef Menu Management App** is a React Native mobile application developed for the **MAST5112 Portfolio of Evidence (PoE)**.

The application provides a simple and intuitive way for a private chef to manage a digital menu. Users can add new dishes, browse the menu, filter dishes by course, and remove menu items. The app is designed to demonstrate core React Native development concepts such as component-based architecture, state management, navigation, user input handling, and reusable code.

## рџљЂ Installation

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Christoffel-Private-Chef-Menu-App.git
```

### 2. Open the project

```bash
cd Christoffel-Private-Chef-Menu-App
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the application

If using Expo:

```bash
npx expo start
```

If using React Native CLI:

```bash
npx react-native run-android
```

or

```bash
npx react-native run-ios
```

## рџ“– How to Use

1. Launch the application.
2. The Home screen displays the current menu.
3. Select **Add Dish** to create a new menu item.
4. Enter:

   * Dish name
   * Description
   * Course
   * Price
5. Save the dish.
6. Return to the Home screen to view the updated menu.
7. Use the Filter screen to display dishes from a specific course.
8. Remove unwanted dishes using the delete option.
